## 조동건 / CDGDG

Flutter로 시작했고, 지금은 앱, API, 자동화, AI 에이전트 워크플로우를 묶어
작은 제품을 빠르게 만들고 실제로 굴리는 쪽에 관심이 많습니다.

앱 화면만 보는 것보다 API 연동, 배포, 운영, 자동화까지 이어지는 흐름을 같이 보는 편입니다.
Codex와 Claude Code는 대체 도구라기보다 반복 작업을 줄이고 개발 범위를 넓히는 도구로 사용하고 있습니다.

### 지금 관심 있는 것

**AI Agent Workflow**

Codex / Claude Code로 반복 작업을 맡기고, 결과를 다시 확인하는 개발 흐름을 만들고 있습니다.

**Product Development**

Flutter 앱, Flutter Web, API 연동, 관리자 도구, 릴리스 운영까지 한 흐름으로 다룹니다.

**Automation**

콘텐츠 생성, 브라우저 QA, 앱스토어 배포 보조, 예약 작업처럼 반복되는 일을 자동화합니다.

**Knowledge Graph**

작업 기록과 메모를 Obsidian, 포트폴리오의 브레인 그래프로 정리하고 있습니다.

### AI Agent Activity

로컬 개발 환경에서 수집한 AI 에이전트 작업 기록입니다.
공식 사용량 API가 아니라, 개인 개발 환경의 local usage snapshot에 가깝습니다.
